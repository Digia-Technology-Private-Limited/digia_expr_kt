# Digia Expressions

A powerful and flexible expression evaluator for Kotlin, designed to dynamically process expressions with support for variables, custom functions, and string interpolation.

> **Note**: This package is developed for internal use within Digia UI (Android) and is not intended for general public consumption.

## Key Features

- Evaluate complex mathematical and logical expressions
- Full support for variables and user-defined function calls
- Advanced string interpolation and manipulation capabilities
- Access properties within JSON objects and Maps
- Easily extensible with your own functions and operations
